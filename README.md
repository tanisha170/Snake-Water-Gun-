# 🐍💧🔫 Snake, Water, Gun

A simple **Snake, Water, Gun** game (a fun twist on Rock-Paper-Scissors) built in Python — just a little project made for fun while learning Python.

## How it works

- You choose one of: `Snake`, `Water`, or `Gun`
- The computer randomly picks one too
- Rules:
  - 🐍 Snake drinks Water → Snake wins
  - 💧 Water short-circuits Gun → Water wins
  - 🔫 Gun kills Snake → Gun wins

## Run it

```bash
python game.py
```

You'll be prompted to enter your choice, and the result (win/lose/draw) will be printed.

## Tech

- Pure Python, no external dependencies — just the built-in `random` module.

## Why I made this

A small beginner-friendly project to practice conditionals, dictionaries, and basic game logic in Python.
